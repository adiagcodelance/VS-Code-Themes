# Dev Themes

A collection of 4 beautiful dark themes for Visual Studio Code.

## Themes

### Purple Surface Dark
- Based on custom color palette (#5f34c8 primary, #121212 surface)
- Deep black backgrounds with purple accents
- Modern, sleek design perfect for night coding

### Ubuntu Dark
- Inspired by Ubuntu's iconic orange and dark gray palette
- Uses Ubuntu's signature orange (#e95420) as accent color
- Warm, familiar colors based on Ubuntu terminal

### Moon Phases
- Cool oceanic color palette (#212A31, #124E66, #748D92, #D3D9D4)
- Deep teal accents with blue-gray highlights
- Professional, calming aesthetic

### Solar Future
- Teal and warm copper palette (#2C3531, #116466, #D9B08C, #FFCB9A)
- Balanced mix of cool teals and warm peachy tones
- Modern, energetic feel

## Installation

### Local Development

1. Copy this folder to your VS Code extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions`
   - macOS/Linux: `~/.vscode/extensions`

2. Reload VS Code
3. Go to File > Preferences > Color Theme (or press `Ctrl+K Ctrl+T`)
4. Select from:
   - **Purple Surface Dark**
   - **Ubuntu Dark**
   - **Moon Phases**
   - **Solar Future**
